# CSS 列布局

```CSS
.column-style {
  /* 列数 */
  -moz-column-count: 3;
  -webkit-column-count: 3;
  column-count: 3;

  /* 列间距 */
  -moz-column-gap: 40px;
  -webkit-column-gap: 40px;
  column-gap: 40px;

  -moz-column-rule: 4px outset #f3f3f3;
  -webkit-column-rule: 4px outset #f3f3f3;
  column-rule: 4px outset #f3f3f3;
}

```
